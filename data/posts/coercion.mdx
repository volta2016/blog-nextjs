## Coerción

Probablemente pueda que conozcas, pero si no conodes markdown sirve para dar un estilo personalizado
a tus textos, personalmente lo ocupo para mis apuntos de programación con JavaScript y para apuntes
de sitios de estudios donde comunmente guardo url, también para mis propios apuntes de estudios.

Si bien siempre para los desarrolladores fue mas complicado incluir html en markdown has surgido
soluciones.

¿Que es MDX?

MDX es un formato de archivos donde puedes personalizar el contenido de texto. Permite extender Markdown con código JSX.
Para el tema de SEO de un blog ayuda mucho gracias al tema de la semantica html5 y contenido a la hora de publicar

¿ Diferencias entre markdown y mdx ?
Los 2 tienen markdown. MDX tiene componentes de react, lo que hace es buscar esos componentes y transpilar a html
automáticamente. Cuando lo haces con Markdown viene ese string que es el lenguaje de marcado.
